# Flowers Cards

Flowers Cards is a visual and practical floral atlas for a beginning florist.

The first direction is an archival botanical poster paired with readable index cards: beautiful enough to feel like a designed product, practical enough to use while learning real flowers, seasonality, care, storage, vase life, sourcing regions, and composition roles.
